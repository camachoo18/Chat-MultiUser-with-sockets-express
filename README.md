# Chat MultiUser con Sockets y Express

Una aplicación de chat en tiempo real para múltiples usuarios utilizando **Node.js**, **Express**, **Socket.IO** y **SQLite3**.

## 📋 Descripción

Aplicación web que permite a múltiples usuarios conectarse simultáneamente y comunicarse en tiempo real. Los mensajes se persisten en una base de datos SQLite, permitiendo que nuevos usuarios vean el historial completo al conectarse.

## 🚀 Características

- ✅ Chat en tiempo real con múltiples usuarios
- ✅ Asignación automática de nombres de usuario (User_XXX)
- ✅ Persistencia de mensajes en SQLite
- ✅ Historial de mensajes disponible al conectarse
- ✅ Interfaz simple y responsive
- ✅ Comunicación bidireccional con WebSockets (Socket.IO)
- ✅ Sesiones de usuario con Express Session

## 📦 Requisitos

- Node.js (v12 o superior)
- npm
